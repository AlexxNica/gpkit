GPkit Documentation
*******************

Welcome!

GPkit is a Python package for defining and manipulating
GP (geometric programming) models,
abstracting away the backend solver.
Supported solvers are
`MOSEK <http://mosek.com>`_
and `CVXOPT <http://cvxopt.org>`_.
