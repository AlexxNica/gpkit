Citing GPkit
************

If you use GPkit, please cite it with the following bibtex::

    @Misc{gpkit,
          author={Edward Burnell and Warren Hoburg},
          title={GPkit},
          howpublished={\url{https://github.com/convexopt/gpkit}},
          year={2015},
          note={Version 0.3.4}
         }
